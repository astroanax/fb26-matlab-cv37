# Formula Bharat 2026 - Vehicle Dynamics Modeling

**Team Unwired, NIT Calicut**

Vehicle dynamics simulation for CX04 ICE Formula Student car using MATLAB Simscape Multibody.

## Contents

- `Vehicle_data_C37_Unwired.m` - Vehicle parameters
- `C37_Gmax_170_50R13.tir` - Tire model
- `generate_acceleration_track.m` - Track generation
- `run_acceleration_simulation.m` - Simulation script
- `Team_Unwired_VehicleDynamics_SUBMISSION.pdf` - Competition report
- `results/` - Simulation plots
- `pics/` - Model screenshots
- `sensor_code/` - IMU data acquisition
- `validation/` - Validation pipeline
- `docs/` - Documentation
